ThriftBooks

Connect, Share and Discover Books with Ease.

Visit the GitHub repository

This was my capstone project for my Meta University Engineering Internship. ThriftBooks can be used by the users to sell, buy and rent used books. App is mainly targeted for the people who generally love reading books and like sharing it with people of similar interests.People can follow each other, comment and message to reach out to buy books.

Understanding the user stories

To give a background about the project, understanding the user stories let’s look at what users are able to achieve by using this app.

  • Users can signup and login to post books
  • Pick their favorite genre of the book [fictional/non-fictional/educational]
  • Compose page to post their books where either they can choose pictures from the library or click pictures with their camera.
  • Search for different keywords along with different filters (title of books, author, release date)

Setting up the project

  • Android Studio with different libraries for UI
  • SDKs from Facebook and Google for easy sign up and loggin in
  • API : Google Books
  • Google Cloud Messaging for Real time messaging
